# 復原{#restoration}



在乾淨的伺服器上，還原程式如下：

* 在安裝與設定的作業系統（網路）上，
* 安裝協力廠商應用程式：網頁伺服器、JDK （如有必要）、
* 使用與來源系統相同的組建安裝Adobe Campaign二進位檔，
* 複製組態檔、追蹤記錄檔和重新導向檔案，
* 建立及重建資料庫，
* 啟動Adobe Campaign。

如需詳細資訊，請參閱 **安裝指南**.
