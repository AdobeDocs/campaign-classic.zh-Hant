# 復原{#restoration}

![](../../assets/v7-only.svg)

在乾淨的伺服器上，恢復過程如下：

* 安裝和配置的作業系統（網路）上，
* 安裝第三方應用程式：Web伺服器、JDK（如有必要）、
* 安裝與來源系統具有相同組建的Adobe Campaign二進位檔，
* 複製配置檔案、跟蹤日誌和重定向檔案，
* 建立和重建資料庫，
* 啟動Adobe Campaign。

如需詳細資訊，請參閱 **安裝指南**.
