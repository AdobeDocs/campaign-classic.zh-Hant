# 復原{#restoration}



在乾淨的伺服器上，還原程式如下：

* 在安裝與設定的作業系統（網路）上，
* 安裝協力廠商應用程式：網頁伺服器、JDK （若有需要）、
* 安裝與來源系統具有相同組建的Adobe Campaign二進位檔，
* 複製組態檔、追蹤記錄檔和重新導向檔，
* 建立和重建資料庫，
* 啟動Adobe Campaign。

如需詳細資訊，請參閱&#x200B;**安裝指南**。
