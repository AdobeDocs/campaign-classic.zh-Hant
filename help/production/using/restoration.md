# 復原{#restoration}

在乾淨的伺服器上，恢復過程如下：

* 安裝和配置的作業系統（網路）上，
* 安裝協力廠商應用程式：Web伺服器、JDK（如有必要）、
* 安裝與來源系統具有相同組建版本的Adobe Campaign二進位檔案，
* 複製組態檔、追蹤記錄檔和重新導向檔案，
* 建立和重建資料庫，
* 啟動Adobe Campaign。

For more information, refer to the **Installation Guide**.
