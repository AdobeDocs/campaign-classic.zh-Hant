# 前言{#foreword}

本檔案詳細說明Adobe Campaign的生產程式：配置、監控、升級過程、資料處理、資料庫維護過程和建議。 本檔案結尾提供疑難排解章節。

生產程式必須謹慎執行，其影響必須事先充分考慮，程式必須嚴格執行。 只能由專家使用者執行。 我們強烈建議您在開始任何生產程式（尤其是升級後步驟）之前，先與Adobe Campaign聯絡。
