# 時間限制{#time-constraint}

**時間限制**&#x200B;活動可讓您延遲執行任務或放棄任務。

輸入活動的標籤，並指定必須暫停工作流任務的時間範圍。

選取&#x200B;**[!UICONTROL Try again later if outside of execution period]**&#x200B;選項後，可讓您在執行時間範圍之外重新啟動任務。 如果希望在暫停後永久放棄工作流操作，請取消選擇此選項。
