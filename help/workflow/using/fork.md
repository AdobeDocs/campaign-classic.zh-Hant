# 分支{#fork}

叉子可讓您同時啟動其所有對外活動。

例如，您可以在內容建立和傳送傳送自動化的上下文中使用活動，以便同時啟動目標計算和內容建立。 [本節](../../delivery/using/automating-via-workflows.md#creating-the-delivery-and-its-content)中提供專用的使用案例。

若要設定活動，請將其開啟，然後定義所要之出站轉場的數目和標籤。
