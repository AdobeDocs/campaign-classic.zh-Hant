# 警報{#alert}

**警報**&#x200B;活動向一組運算子發送消息。 其運作方式與核准活動相同，但在此情況下不會預期有任何回應。

![](assets/edit_alerte.png)

警報不具持續性，因此無法從主控台顯示。 指派群組的運算子必須有完整的電子郵件地址，才能接收通知。 此活動的配置與&#x200B;**Approval**&#x200B;的配置類似。 用於警報運算子的預設傳送範本為「alertAssignee」。
