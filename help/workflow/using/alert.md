# 警報{#alert}



一個 **警報** 活動會傳送訊息給一組操作者。 其運作方式與核准活動相同，但在此情況下預期不會有任何回應。

![](assets/edit_alerte.png)

警示不是持續性的，因此不會從主控台顯示。 指派群組的操作者必須有完整的電子郵件地址才能接收通知。 此活動的設定類似於 **核准**. 用於警示操作員的預設傳遞範本為「alertAssignee」。
