# 工作流活動{#wf-activities}

![](../../assets/common.svg)

本節介紹所有可用活動。 根據建立/編輯工作流的節點或上下文，可用的活動可能會有所不同。 例如，在市場活動中建立的工作流具有特定於渠道的交付活動。

工作流活動按類別分組。 同時有四個頁籤。

在市場活動工作流中， **[!UICONTROL Events]** 頁籤替換為 **[!UICONTROL Deliveries]** 頁籤。 此頁籤中的活動在 [行動活動](about-action-activities.md) 的子菜單。
