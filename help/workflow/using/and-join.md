# 合併連結{#and-join}

只有在激活所有入站轉換（即在所有先前活動完成時）時，連接才會觸發其出站轉換。 這可讓您在繼續執行工作流程之前，先確定某些活動已完成。

通過在活動的入站轉變中選擇主集來確定活動的出站發送人口。

出站轉移只能包含其中一個入站轉移人口。 如果未配置活動，則出站轉移將隨機選擇一個入站人口族群。
