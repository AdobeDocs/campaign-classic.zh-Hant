# 清除事件{#purging-events}

您可以使用部署嚮導來配置資料在資料庫中儲存的時間。

事件清除由&#x200B;**[!UICONTROL Database cleanup]**&#x200B;工作流自動執行。 此工作流程會清除在執行例項上接收和儲存的事件，以及封存在控制例項上的事件。

根據需要使用箭頭來更改清除設定。
