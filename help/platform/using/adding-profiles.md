# 新增輪廓{#adding-profiles}



在大多數情況下，設定檔會[透過工作流程](../../platform/using/import-export-workflows.md)匯入至Campaign。 不過，您可以按一下&#x200B;**[!UICONTROL Create]**&#x200B;按鈕，直接從介面新增一或多個設定檔。 然後將它們新增至資料庫。

![](assets/s_ncs_user_profile_add.png)

>
