# 環境概覽{#environments-overview}



本章詳細說明如何在「互動」中管理優惠方案環境。

管理和使用選件涉及多個不同的使用者設定檔。 因此，第一個步驟包括確保這些設定檔存在，如果沒有，則建立它們。

>[!IMPORTANT]
>
>下列設定需要由執行 **技術管理員**.
