# 環境概觀{#environments-overview}

本章詳細說明如何在Interaction中管理選件環境。

管理和使用選件時會涉及數個不同的使用者設定檔。 因此，第一步包括確保這些配置檔案存在，如果不存在，則建立它們。

>[!CAUTION]
>
>以下配置需由技術管理員 **執行**。
