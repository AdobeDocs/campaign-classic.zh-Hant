# 識別表單{#identifying-a-form}

輸入表單可讓您從Adobe Campaign用戶端主控台編輯與資料結構關聯的例項。

表單由其名稱和命名空間來識別。

表單的識別索引鍵是由namespace和名稱(以冒號(&#39;:&#39;)分隔)組成的字串(例如：「cus:contact」)。
