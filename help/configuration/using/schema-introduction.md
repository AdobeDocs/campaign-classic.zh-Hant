# 結構描述參考 {#schema-reference}

編輯結構描述時，可以使用根據來源結構描述(xtk：srcSchema)的核准系統。 使用「資料庫結構更新……」精靈更新資料庫時，也會發現一些錯誤。

依預設，在Adobe Campaign結構描述中，所有布林值型別屬性都是「false」。 若要啟用它們，您必須在結構描述中指定屬性，並將其值設為「true」。
