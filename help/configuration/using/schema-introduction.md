# 結構參考 {#schema-reference}

編輯架構時，可使用以來源架構(xtk:srcSchema)為基礎的核准系統。 使用「資料庫結構更新……」更新資料庫時，也可能發現一些錯誤。 嚮導。

依預設，在Adobe Campaign結構中，所有布林類型屬性都為「false」。 若要啟用這些屬性，您必須在結構中指定屬性，並將其值設為「true」。
