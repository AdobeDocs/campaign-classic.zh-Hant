# `<methods>` 元素  {#methods--element}

## 內容模型{#content-model-11}

## 屬性{#attributes-11}

無

## 父代{#parents-11}

## 子項{#children-11}

## 說明 {#description-11}

此元素可讓您定義`<method>`元素。 聲明方法是強制的。

## 屬性說明{#attribute-description-11}

此元素沒有屬性。

...// definition of one or more <method
