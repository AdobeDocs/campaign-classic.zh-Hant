# 元素和屬性 {#elements-and-attributes}

編輯架構時，可使用基於源架構(xtk:srcSchema)的批准系統。 使用「資料庫結構更新……」更新資料庫時，也可能發現一些錯誤。 嚮導說明。

依預設，在Adobe Campaign結構描述中，所有布林類型屬性都是「false」。 若要啟動這些屬性，您必須在結構中指定屬性，並將其值設為&quot;true&quot;。
