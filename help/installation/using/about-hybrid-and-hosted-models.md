本節將介紹不同的代管 [模型](../../installation/using/hosting-models.md)。

若要進一步瞭解部署模式之間的主要差異，請參閱 [本文章](https://helpx.adobe.com/tw/campaign/kb/acc-on-prem-vs-hosted.html)。
